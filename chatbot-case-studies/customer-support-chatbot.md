# Customer Support Chatbot

## Project Objective
To reduce incoming support tickets by resolving common customer issues through automated chatbot workflows.

## Problem
Support team was receiving a high volume of repetitive customer questions which increased resolution time.

## Solution
Designed a chatbot that handles:
- Order tracking
- FAQ responses
- Ticket creation
- Live agent handoff for complex issues

## Workflow
1. Customer opens chatbot
2. Selects issue category
3. Bot provides instant resolution using APIs
4. Escalates to agent if required

## Skills Demonstrated
Chatbot design | Customer experience | Process optimization | Technical communication
